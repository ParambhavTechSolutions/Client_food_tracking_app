# Cloud Kitchen Apps 

A comprehensive Cloud Kitchen Management System built with Next.js 14, Tailwind CSS, and TypeScript.

## 🚀 Key Features

*   **Multi-Tenant Architecture**: Built for scale with Franchise support.
*   **Customer App**: QR Code ordering, dynamic menu, cart, and mock payments.
*   **Kitchen Display System (KDS)**: Real-time order management (Kanban style).
*   **Admin Dashboard**: Menu management, price updates, and availability toggles.

## 🔗 Quick Links (Localhost)

Once the server is running (`npm run dev`), you can access the various modules here:

| Module | URL | Description |
| :--- | :--- | :--- |
| **👩‍🍳 Kitchen Display** | [http://localhost:3000/kitchen/store_123](http://localhost:3000/kitchen/store_123) | Chef's view to manage incoming orders. |
| **📱 Customer Menu** | [http://localhost:3000/menu/store_123/1](http://localhost:3000/menu/store_123/1) | Scanning a QR code lands here (Table 1). |
| **🛠️ Admin Dashboard** | [http://localhost:3000/admin/store_123](http://localhost:3000/admin/store_123) | Manager's view to update menu items. |

## 🛠️ Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## 🏗️ Tech Stack

*   **Framework**: Next.js 14 (App Router)
*   **Styling**: Tailwind CSS
*   **Language**: TypeScript
